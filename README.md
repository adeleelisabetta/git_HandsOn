The code named seqClass is useful to individuate if a sequence it's DNA or RNA, it takes to account 
if it's written in upper o lower case, changing everything in uppercase. 
Based on the presence of T or U, it's able to establish if it's DNA or RNA respectively.
It's also able to identify any motif inside the string.
It's also written to stop if it doesn't recognise any nucleic acid.
